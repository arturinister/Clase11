Wireframing

Es una técnica para diseñar rápidamente sitios web, con la característica que es un diseño sencillo. Su propósito es ver rápido un diseño inicial.
Es un framework de HTML, CSS y JS que ayuda a los diseñadores web a crear rápidamente estilos. Tiene elementos predefinidos que pueden modificables en otro archivo CSS
